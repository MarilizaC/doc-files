- **Using the Subutai Components**  
 Aside from the products, learn about the software components that enable easy integration, management, and access to Subutai’s   features.  

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; **E2E Plugin** - facilitates working with peers by handling PGP key generation and management of PGP-based authentication and encryption. Learn more.  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; **P2P Daemon** - program that runs in the background to handle intra-environment  connectivity and communication between containers, primarily enabling peers to join the swarm. Learn more.  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; **Control Center** - convenient desktop application for setting up peers, components, and environments, with seamless Bazaar and P2P Daemon integration. Learn more.  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; **Management Console** - the application for peer administration, with additional functionality for peer registration and creation of cloud environments. Learn more.
